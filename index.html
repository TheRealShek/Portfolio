About Me

Passionate about creating innovative solutions

A DevOps Engineer passionate about building scalable, secure, and automated infrastructure. I specialize in CI/CD with Jenkins and GitLab, containerization with Docker, and Kubernetes orchestration on EKS.

My work revolves around Infrastructure as Code using Terraform and Ansible, with a strong focus on AWS cloud architecture. I implement DevSecOps practices, manage secrets and IAM roles, and monitor systems using Prometheus, Grafana, and Loki. I'm driven by automation, reliability, and continuous improvement in everything I build.
